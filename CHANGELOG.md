## [1.0.0] - 2026-04-10

### Added
- `TreeDropdown` Filament form component with fluent builder API
- `group()` and `item()` methods for building nested tree structures
- `tree()` method for passing raw arrays
- Configurable `labelKey`, `valueKey`, `childrenKey`
- Optional `searchable()` filtering
- `FilamentTreeDropdownPlugin` class for Filament v3 panel registration
